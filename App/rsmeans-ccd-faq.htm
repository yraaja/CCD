CCD Frequently Asked Questions

  • CCD Version 7.3
    • I receive "Internal Error 2738" during Setup.
      • InstallShield uses the Microsoft Scripting during installation if the Data Access Components need to be upgraded. If you are using an Anti-Spam product such as Microsoft Anti-Spam, please make sure that it allows Scripts to run or temporarily disable it while installing CCD.
    • During Installation or Uninstallation, I receive a warning box (with no text) that references Microsoft Outlook.
      • This is an InstallShield generated warning. Please close Microsoft Outlook during the Installation or Uninstallation of CCD. After closing Outlook, click "Retry" to continue.
    • When I try to send a fax, it sends as an e-mail instead.
      • You must change the order that Outlook resolves contacts. In Outlook, select Address Book from the Tools menu. In Address Book, select Tools then Options. Select Contacts and use the arrow buttons to move it to the top of the list. Click OK.
    • I receive Error 1002-462 "The Remote Server Machine does not exist" when trying to send a Fax or E-Mail.
      • You must have Outlook 2002 installed on your workstation to use the Fax or E-Mail features. Contact the Help Desk to have Microsoft Office/Outlook 2002 installed on your PC.
    • I receive Error 64 "Application-defined or object-defined error" when trying to send a Fax or E-Mail.
      • Windows XP does not support writing temporary files to the application folder under non-administrator user accounts. If you are running Windows XP, you should upgrade to CCD version 7.3 SP1 to correct this problem.
    • Where can I find out the status of my faxes?
      • In Microsoft Outlook's Folder List, expand Mailbox - RSMeans and click the Inbox. You will see a message similar to "Your fax has been successfully sent to...". If you do not see the Outlook Folder list, open the View menu and select Folder List. If you do not have the RSMeans Mailbox listed in your Outlook Folders, please see the next question or contact the Help Desk.
    • How do I add the RSMeans Mailbox to my Outlook Folder List?
      • In Microsoft Outlook, open the Tools menu and click E-Mail Accounts. Select "View or change..." and click Next. Select "Microsoft Exchange Server" and click the Change button. Click the More Settings button. On the Advanced tab, click Add. Enter "RSMeans" and click OK. "Mailbox - RSMeans (RBI-US)" should now be listed under Additional Mailboxes. Click OK, then Next and Finish.
    • The fax status message says "Undeliverable: Request for Quote" from System Administrator.
      • This message is produced when the Info Source recipient is missing either a last name or a first name. Workaround: Update the Info Source with both a first and last name to correct this problem. (Note: This problem is corrected in CCD 7.4 or later.)
    • The fax status message says "Fax to <name> abandoned after 5 attempts."
      • Click the message to view the reason the fax was abandoned, for example "remote end busy" or "human answer detected". Correct the problem if possible and try again.
    • The fax status message says "Imaging error encountered, fax abandoned (From: RSMEANSBOS To: ...)".
      • Click the message to open it. Click in the Subject box and scroll to the right or press the [End] key on the keyboard to read the Error message. A common example is "Bad fax phone number specified". Double-check the fax number in the Info Source and make sure it is in the format "123-456-7890".
    • The fax status message says "Imaging error encountered, fax abandoned" and nothing else.
      • The Fax Server was unable to convert the message into a fax for an unknown reason. Please try again.
  • CCD Version 7.3 SP1 (7.3.1)
    • What is the difference between 7.3 and 7.3 SP1?
      • 7.3.1 saves temporary files to the Windows user's Temporary Folder instead of the system drive or the Application Folder. This eliminates Error 64 when sending a fax or e-mail from CCD running on Windows XP.
    • Should I upgrade to 7.3 SP1 if I already have 7.3?
      • You only need to upgrade to 7.3 SP1 if you are running Windows XP.

    • I can't select MasterFormat 2004 in the Unit Cost screen.
      • MasterFormat 2004 will be fully supported in CCD 7.5.
    • How do I print all CCI Material/Equipment Exceptions at once?
      • Select the appropriate Geographic Selection and the Quarter. In the Tree, click either the Material or Equipment node to return all data. Click the Preview button. The entire report will be displayed, each ID starting a new page. Click the Print toolbar button to print the report.
  • CCD Version 7.5
    • How do I set which MasterFormat the Unit Cost Maintenance defaults to?
      • Under the Tools menu, select the Settings sub-menu, Default MasterFormat, then choose MasterFormat 1995 or MasterFormat 2004.
    • How do I add a MasterFormat 2004 ID to an existing unit cost (1995 format)?
      • Go to the Unit Cost Maintenance screen. Change the MasterFormat drop-down to "MF-1995" if it's not already selected. Select the Unit Cost ID in the MasterFormat Tree, or use the Search function. Select the Unit Cost line, then click the Go To Unit Cost button. Enter the new ID in the "Unit Cost ID 04" box and click Update.
    • My new Unit Cost line does not have an existing 1995 unit cost. How do I add a new MasterFormat 2004 unit cost?
      • Go to the Unit Cost Maintenance screen. Change the MasterFormat drop-down to "MF-2004" if it's not already selected. To clone an exising line, select a Unit Cost ID in the MasterFormat Tree, or use the Search function, then click Clone; or else click New. Make any required changes or additions. Make sure the "Unit Cost ID 95" box is empty. Click Update.
